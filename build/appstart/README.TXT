AppStart
========================

= How to use AppStart =

Put the start.jar and appstart.properties files together into a folder. Then, put
some .jar files into a subfolder called "lib".
Configure appstart.properties, change the value of app.main.class to the main class 
of the application to run.

Double-click the start.jar and your application is running
